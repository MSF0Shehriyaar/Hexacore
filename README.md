# ⬢ Hexacore (Lumina AI)
### A Unified Multi-Dimensional Intelligence Platform

**Hexacore** is a centralized, self-contained intelligent platform that unifies learning, coding, career development, and content creation into a single, NLP-driven tool orchestration system. It is designed to bridge the gap between academic mastery and professional excellence through six core "Dimensions of Intelligence."

---

## 🚀 The Six Dimensions

### 1. 🎓 Poly Argu-Minds (Global Study Platform)
*   **AI Tutor (Voice-Enabled):** Real-time voice-based AI using browser audio APIs for seamless STT/TTS interaction.
*   **Intelligent Study Groups:** Collaborative environments with embedded AI tools for translation, summarization, and explanation.
*   **Neural Peer Matching:** Connects mentors and peers globally based on cognitive compatibility.

### 2. 💻 CodeForge (Elite Coding Arena)
*   **Sandboxed IDE:** Monaco-based workspace with real-time execution across 24+ languages.
*   **AI Code Assistant:** Intent-driven tools for code generation, debugging, optimization, and explanation.
*   **Execution Console:** Docker-based runtimes providing instant feedback and AI-driven error analysis.

### 3. 📄 Resume Architect (Professional Network)
*   **Semantic Extraction:** Automatically identifies hidden skills from your ecosystem activity.
*   **ATS Optimization:** Real-time scoring against global job descriptions.
*   **Verified Endorsements:** Trust-based network for skill validation and industry connections.

### 4. 💼 Verified Job Network (Career Launchpad)
*   **KYC Verification:** Every company and job post is vetted to prevent scams and ghost jobs.
*   **Direct-to-CTO Messaging:** Secure communication channels between talent and technical leadership.
*   **Blockchain Credentials:** Verifiable candidate history and performance benchmarks.

### 5. 📚 Digital Knowledge Library
*   **Vector Indexing:** Sub-second semantic retrieval of research papers, tutorials, and documentation.
*   **3D Visualization:** Interactive WebGL-powered modules for complex scientific and engineering concepts.
*   **LLM Synthesis:** Automated synthesis of answers from multiple verified sources.

### 6. 🎨 Creative Studio
*   **Content Generation:** Structured output for blogs, social media, and professional emails.
*   **Transformation Pipeline:** Sequential AI actions (Summarize -> Rewrite -> Translate).
*   **Advanced Utility:** Standalone TTS, Idea Generation, and Tone Adjustment engines.

---

## 🛠 Technical Architecture

- **Frontend:** React (TypeScript) + Framer Motion (Immersive UI/UX)
- **Backend:** Spring Boot (Java) (Orchestration & Business Logic)
- **AI Infrastructure:** Custom Python-based NLP Microservice (Intent Classification & Entity Extraction)
- **Database:** PostgreSQL (Structured Data & Logs)
- **Real-time:** WebSockets (Chat & Execution Feedback)

## 🛠 Setup & Installation

### Prerequisites
- Node.js (v18+)
- Java (JDK 17+)
- Python (v3.9+)
- Docker (Optional, for CodeForge execution)

### 1. Frontend Setup
```bash
cd hexacore
npm install
npm run dev
```

### 2. Spring Boot Backend
```bash
cd spring-boot-backend
./mvnw spring-boot:run
```

### 3. NLP Microservice
```bash
cd nlp_service
pip install -r requirements.txt
python main.py
```

## 🛡 Governance & RBAC
Hexacore implements a strict Role-Based Access Control system:
- **Admin:** Full system control and DSA problem management.
- **Professional:** Full access to learning and creative dimensions.
- **Teacher:** Content creation and student guidance.
- **Student:** Guided learning with focused communication boundaries.
